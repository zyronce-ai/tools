# Ecommerce Tools - All-in-One AI Toolkit

AI-powered ecommerce tools for content writing, banner making, keyword research, competitor analysis, and more.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Supabase
- TanStack Query
- React Router v6

## Getting Started

```sh
npm install
npm run dev
```

Dev server runs on http://localhost:8080

## Build

```sh
npm run build
```
